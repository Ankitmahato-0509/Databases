# Employee Database SQL Project

## Project Description

This project creates an Employee database using SQL.
It includes table creation, inserting records, and basic queries to display employee details.

This project is made for DBMS lab practice and my 100 Days of Code challenge.

## Technologies Used

* SQL
* SQL Server Management Studio 2022 (SSMS 22)
* GitHub

## Features

* Create Employee table
* Insert employee records
* View employee data
* Filter employee details using SELECT queries

## How to Run

1. Open SQL Server Management Studio 2022
2. Connect to your SQL Server instance
3. Create a new database
4. Open the file `employee-database-project.sql`
5. Click Execute to run the script
6. Run SELECT queries to view results

## Learning Outcome

* Understanding CREATE TABLE
* Using INSERT INTO
* Writing SELECT queries
* Basic database management practice
